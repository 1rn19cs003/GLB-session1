# ◈ DevHub

DevHub is a modern, open-source platform where developers can discover, collaborate on, and showcase their projects. Built with React and Vite, it features a responsive UI, mocked authentication, user/admin dashboards, and a comprehensive project directory.

## 🚀 Quick Start

Follow these steps to get the project running on your local machine.

### 1. Prerequisites
Ensure you have the following installed on your system:
* **Node.js** (v16.0.0 or higher recommended)
* **npm** (comes with Node.js) or **yarn**

### 2. Installation
Navigate to the project directory and install the required dependencies:

```bash
# Install dependencies
npm install
```

### 3. Run locally
Navigate to the project directory and run :

```bash
# run locally
npm run dev
```
